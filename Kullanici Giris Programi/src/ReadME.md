## Kullanıcı Giriş Programı

Kullanıcıdan kullanıcı adı ve şifre isteyip bilgileri kontrol eden bir java programı yazın.Eğer kullanıcı adı veya şifre yanlış ise kullanıcıya değiştirmek isteyip istemediğini sorun.

Eğer değiştirmek isterse değiştirip tekrar giriş yapmasını sağlayın değiştirmek istemezse programı sonlandırın.
