## Faktoriyel Hesaplayan Program

n bir pozitif tam sayı olmak üzere 1’den n’ye kadar olan doğal sayıların çarpımına n faktöriyel denir ve n! şeklinde gösterilir.

n! = 1 . 2 . 3 . … . (n – 1) . n ‘dir.
0! = 1 olarak kabul edilir.

* 0! = 1

* 1! = 1

* 2! = 1.2 = 2

* 3! = 1.2.3 = 6

* 4! = 1.2.3.4 = 24

* 5! = 1.2.3.4.5 = 120

* 6! = 1.2.3.4.5.6 = 720

* 7! = 1.2.3.4.5.6.7 = 5040
