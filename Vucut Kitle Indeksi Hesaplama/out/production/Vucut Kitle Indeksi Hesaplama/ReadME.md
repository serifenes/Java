## Vücut Kitle İndeksi Hesaplama

Java ile kullanıcıdan boy ve kilo değerlerini alıp aşağıdaki formüle göre kullanıcının "Vücut Kitle İndeks" değerini hesaplayıp ekrana yazdıran programı kodluyoruz.

### Formül

Kilo (kg) / Boy(m) * Boy(m)

### Çıktısı

Lütfen boyunuzu (metre cinsinde) giriniz : 1,72

Lütfen kilonuzu giriniz : 105

Vücut Kitle İndeksiniz : 35.49215792320173
