## Arkadaş Sayıları Bulan Program

Klavyeden girilen 2 sayının arkadaş sayı olup olmadığını kontrol edip ekrana yazdıran programın kodlarını yazıyoruz.

* Arkadaş sayılar : İki sayı birbirinin kendisi hariç bölenleri toplamına eşitse bu sayılara arkadaş sayılar denir. Örnek: 220 ve 284 220: 1 + 2 + 4 + 5 + 10 + 11 + 20 + 22 + 44 + 55 + 110 = 284  284: 1 + 2 + 4 + 71 + 142 = 220
